# DupeMod для Minecraft 1.21.1

## Описание
Клиентский мод для NeoForge, который позволяет дюпать предметы в руке при нажатии на клавишу **=**.

## Установка

1. Убедитесь, что у вас установлен **NeoForge 1.21.1-51.0.0** в лаунчере
2. Скачайте готовый JAR файл из релизов
3. Поместите его в папку `.minecraft/mods/`
4. Перезагрузите игру

## Компиляция из исходников

```bash
git clone https://github.com/iasper151maglot/dupemod.git
cd dupemod
.\gradlew build
```

Готовый мод будет в `build/libs/dupemod-1.0.0.jar`

## Использование

- Возьмите предмет в руку
- Нажмите клавишу **=**
- Предмет будет дюпнут!

## Совместимость

✅ Работает с другими модами
✅ Работает на ванильных серверах (клиентский мод)
✅ Minecraft 1.21.1 + NeoForge
